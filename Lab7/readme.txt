Вопросы:
1. Каким образом можно убедиться в линейной зависимости данных?
 Можно убедиться, проанализировав поведение точечного графика, построенного
с помощью функции plot(). Для определения линейной зависимости математически
следует найти корелляцию между нашими данными с помощью функции cor.

2. В чем состоит задача метода наименьших квадратов?
 Задача сводится к нахождению оптимальных параметров линейной регрессии, при 
которых расстояние между вектором восстановленных значений и вектором фактических
значений минимально, другими словами сумма квадратов ошибок минимальна.

3. С помощью какой команды реализуется метод наименьших квадратов?
 С помощью команды lm(эндогенная переменная ~ экзогенная переменная)

4. Эндогенные и экзогенные переменные - что это?
 Эндогенная переменная = зависимая переменная = переменная отклика - переменная,
которая подвергается воздействию и меняется в изучаемой экономической модели.
 Экзогенная = независимая переменная = поясняющая переменная - это переменная в
экономической модели, которая влияет на другие переменные в модели, но сама не 
изменяется.

5. Команда attributes(), что делает и для чего нужна?
 Она позволяет получить всю информацию (атрибуты), которая хранится в переменной.

6. Какие есть способы для нахождения остатков?
 1) через формулу с помощью переменных коэффициентов
fit$coefficients[[2]]*year+fit$coefficients[[1]]
 2) С помощью команды residuals
 3) с помощью аттрибута переменной: fit$residuals

7. Какая функция отвечает за получение результатов F-теста, а также других тестов?
 Функция summary()

8. Какая функция используется для построения графика регрессии на одном участке
с точечным графиком? 
 Функция abline()
